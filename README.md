# news_app
News app developed in Flutter 
## Features included
-  Dart
-  flutter
-  bloc architecture
-  Drift for local Database
-  API Integration using Retrofit and Dio
-  Offline fallBack in case of internet connectivity issue
-  Loading and error handling in case of API calls and errors
-  Offline Support
